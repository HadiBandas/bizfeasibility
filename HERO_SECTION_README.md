A world-class, dynamic hero section with Apple-inspired design, featuring lively micro-interactions, subtle motion effects, and Matrix-style data stream animation. Available in both HTML/CSS/JS and React + Framer Motion versions.

## üéØ Animation Details

### Entry Animations
1. **Left Column**: Fade in from bottom (0.2s delay)
2. **Title**: Fade in from bottom (0.3s delay)
3. **Subtitle**: Fade in from bottom (0.4s delay)
4. **Actions**: Fade in from bottom (0.6s delay)
5. **Right Column**: Slide in from right (0.5s delay)

- **Widget Hover**: Lift 12px, scale 1.02, enhanced shadow
